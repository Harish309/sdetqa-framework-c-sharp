How to Add to Visual Studio - Manual copy
1. In Explorer, open "%userprofile%\Documents\Visual Studio 2017\Templates\ItemTemplates\Visual C#"
2. Copy the "Test Automation" folder from the repo to your Documents

How to Add to Visual Studio - Symlinks
  Use symlinks to point to the files rather than copying them. This has the added benefit of always staying up to date with the repo.
